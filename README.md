# cyberpunk-ui
cyberpunk-ui base Vanilla.js

## Documents

**Tabs Example**
```html
<div class="tabs">
    <ul class="tabs-nav">
        <li>Tab 1</li>
        <li>Tab 2</li>
        <li>Tab 3</li>
    </ul>
    <ul class="tabs-panes">
        <li>Content 1</li>
        <li>Content 2</li>
        <li>Content 3</li>
    </ul>
</div>
```
```javascript
new Tabs({
    element: '.tabs',
    initTab: 0,
});
```
